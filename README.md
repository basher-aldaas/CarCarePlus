# CarCarePlus

CarCarePlus is a web application project built with Laravel.
1.تسجيل دخول المستخدم العادي يكون بطريقة طبيعية 
2.تسجيل دخول الشركة يفوت المستخدم ويحط معلومات حسابه وشركته يتم انشاء حساب له وايقاف تفعيله تلقائيا بنتظار رد السوبر ادمن للتفعيل 
3.تسجيل دخول الورشة  يفوت المستخدم ويحط معلومات حسابه والورشة يتم انشاء حساب له وايقاف تفعيله تلقائيا بنتظار رد السوبر ادمن للتفعيل
